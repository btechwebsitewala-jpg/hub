# Diagnostics Hub

A modern pathology lab and diagnostics center management system.

## Project info

This project provides a comprehensive platform for managing diagnostic tests, patient records, and appointments.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm i
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
